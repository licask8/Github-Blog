Durante este desafio, desenvolvemos uma aplicação que utiliza a API do GitHub para buscar issues de um repositório, dados do perfil e exibir elas como um blog.


# Nesse projeto foi utilizado
* Vite
* Styled Components
* Typescript
* React Router Dom
* React Hook Form
* Axios
* API do Github
